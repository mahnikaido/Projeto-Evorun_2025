# 🚀 Evorun — Transform Your Routine into an Epic Journey

Evorun is a gamified digital platform that transforms physical activities such as running and walking into an epic and engaging experience.

The project combines storytelling, technology, and motivation to make physical exercise more enjoyable, accessible, and meaningful — especially for young people aged 18 to 30.

---

## 🎯 Purpose

Encourage consistency in physical exercise through an interactive experience based on:

- Missions
- Rewards
- Immersive narratives
- Personal progression

---

## 🛠️ Main Features

- Personalized narrative-based onboarding
- Weekly missions with motivational rewards
- Narrative audio during workouts
- Progression system with medals, levels, and collectible items
- Intuitive dashboard to track user progress
- User-friendly interface with motivating visual design

---

## 💻 Technologies Used

- React Native + Expo
- Node.js
- Firebase *(coming soon)*
- Figma *(UI/UX prototyping)*

And lots of energy and coffee ☕
